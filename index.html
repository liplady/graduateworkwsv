О сайте HTML для новичков CSS и дизайн

О работе

В качестве дипломной работы по веб-разработке я создала этот сайт, демонстрирующий мои практические навыки создания веб-сайтов с использованием HTML, CSS и JavaScript. Он предлагает пошаговые инструкции, примеры кода и адаптивный дизайн, чтобы показать, как сделать сайт доступным на любом устройстве. Сайт также включает полезные ресурсы для тех, кто хочет углубиться в веб-разработку.

Дизайн

Цветовая палитра

Сине-серая цветовая палитра выбрана мной для создания профессионального, современного и удобного для чтения веб-сайта. Сочетание этих цветов обеспечивает хороший контраст и комфортное восприятие информации для пользователей, при этом отражая технологическую направленность дипломной работы по веб-разработке.

Типографика

Основной шрифт: Questrial

Шрифт для заголовков: Outfit

Основной текст сайта набран шрифтом Questrial, который обеспечивает хорошую читаемость и нейтральный тон. Заголовки оформлены шрифтом Outfit, который, по моему мнению, является более выразительным и современным. Этот контраст помогает визуально разделить контент, делая сайт более организованным и удобным в использовании.

Технические детали

Мой веб-сайт разработан с применением HTML, CSS и JavaScript для создания структуры, стилизации и добавления интерактивности. Перед этапом разработки интерфейс сайта был спроектирован в Figma. Figma использовалась для создания макетов, что позволило оптимизировать процесс разработки и обеспечить соответствие дизайна требованиям к дипломной работе.

Результаты

Результатом дипломной работы является полноценный веб-сайт, который может служить полезным ресурсом для веб-дизайнеров, желающих освоить или улучшить свои знания HTML и CSS. Данный проект демонстрирует навыки разработки веб-сайтов с использованием HTML, CSS и понимание принципов веб-дизайна. В дальнейшем сайт может быть дополнен новыми разделами (например, JavaScript), а также расширен разделом с практическими уроками и примерами.
